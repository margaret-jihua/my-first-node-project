# Setting up a Node project

1. Create a new folder for the first node project
``` 
mkdir my-first-node-project
```

2. Initialize Node inside the project folder
``` 
npm init -y
```

3. Make entry point
``` 
touch index.js
console.log("Hello World") 
```

4. Run program
``` 
node index.js 
```

## install globely:  `nodemon`
```
npm i -g nodemon
```
